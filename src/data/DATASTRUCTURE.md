# Data Structure

For each creature we should structure the json object model to be:

```json
{
  "name": "string",
  "kingdom": "string",
  "class": "string",
  "location": [
    {
      "zone": "string",
      "x": 100,
      "y": 100,
      "z": 100
    }
  ],
  "image": "string",
  "notes": "string"
}
```

It should be organized alphabetically.
